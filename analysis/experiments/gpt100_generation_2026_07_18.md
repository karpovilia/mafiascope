# Догенерация gpt-4o-mini корпуса до 100 игр (gpt4omini100)

Дата: 2026-07-18. Задача: довести корпус полных нефорковых gpt-4o-mini игр с 28 до ровно 100, зарегистрировать `gpt4omini100`, пересчитать ключевые метрики. Сырые числа: `gpt100_generation_2026_07_18.json` (тот же каталог). Статья НЕ правилась; `gpt4omini28` остаётся пинованным подмножеством для чисел статьи.

## 0. Парити-проверка перед генерацией

Конфиг `mafia/configs/config_chatgpt_corpus.yaml` сверен с фактической игрой пинованного корпуса (`16a6b166`, одна из 26 стандартных): все 5 проб (`role_assessment`, `suspicion_ranking`, `social_map`, `personality_profile`, `planned_action`) после подстановки значений совпали с логом **посимвольно**; порядок проб, max_tokens проб (960/960/960/1024), параметры бэкенда (gpt-4o-mini, max_tokens 400, timeout 90, temperature 0.7), язык en, 7 мест gpt-4o-mini — идентичны. `game.snapshots: true` уже стоял в конфиге. Правок конфига не потребовалось.
